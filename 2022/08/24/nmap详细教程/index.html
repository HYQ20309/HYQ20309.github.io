nmap详细教程

发表于2022-08-24|更新于2022-08-26
|阅读量:

nmap简介

Nmap是一个开源、免费的网络探测工具，适用于windows、linux、mac等多种操作系统。能够快速扫描大型网络或者单个主机，可以发现网络上有哪些主机，这些主机提供什么服务，服务运行在什么操作系统（包括版本系统），扩大攻击范围。

用途

1.识别目标网络内活跃的主机。
2.推断主机的操作系统。
3.扫描主机端口
4.利用自带脚本检测漏洞

端口

什么是端口

1.物理端口：也称接口，如usb接口
2.逻辑端口：一般指TCP/IP协议中的端口

端口的数量

0-65535（共65536个）

端口状态

1.Opend 端口开放
2.Closed 端口关闭
3.Filtered 端口被过滤（由于防火墙、路由器规则等阻止到达端口）
4.openfiltered 开放或者过滤的，无法确定端口是开放还是被过滤的
5.closedfiltered 关闭或者被过滤的，nmap不能确定端口是关闭的还是被过滤的。

nmap基本命令

1.nmap IP 常用扫描指令，扫描较粗略
2.nmap -p 端口号 ip 指定端口扫描
3.nmap -r ip 随机扫描端口
4.nmap -v 显示扫描过程
5.nmap -Pn 跳过主机发现过程直接端口扫描 （目标禁ping时）
6.nmap -O 探测主机操作系统
7.nmap -A 全面扫描，包括一切探测
8.nmap -sn IP 检查目标是否联网
9.nmap ip1 ip2 ip3 扫描多个目标
10.nmap -sS 目标IP 半连接扫描，迅速且隐蔽
11.nmap -sT 目标IP 全连接扫描
12.nmap -sL 目标IP 反向域名解析
13.nmap -sV 查看服务版本
14.nmap -sC 查看服务信息
15.nmap –script=discovery IP 信息挖掘
16.nmap -iL 文件 读取文件
17.nmap –script=dos IP 拒绝服务攻击
18.nmap –script=exploit IP 利用已知漏洞攻击
19.nmap –script=external IP 利用第三方的数据库或资源进行信息收集或攻击
20.nmap –script=fuzzer IP 进行模糊测试，发送异常的包到目标机，探测出潜在漏洞
21.nmap –script=safe IP 检查系统安全
22.nmap –script=vuln IP 检查常见漏洞
23.nmap -oX 1.txt IP 将扫描结果另存为1.txt
